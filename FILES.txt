INSTALLATION QUICK REFERENCE
=============================

Windows:
  Copy bitmagnet.py to:
  %LOCALAPPDATA%\qBittorrent\nova3\engines\

Linux:
  Copy bitmagnet.py to:
  ~/.local/share/qBittorrent/nova3/engines/

macOS:
  Copy bitmagnet.py to:
  ~/Library/Application Support/qBittorrent/nova3/engines/

Then restart qBittorrent and enable "Bitmagnet" in Search tab.

REQUIREMENTS
============
- qBittorrent 4.1.0+
- Python 3.7+
- Bitmagnet instance (local/remote)
