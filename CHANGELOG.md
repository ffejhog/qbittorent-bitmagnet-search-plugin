# Changelog

All notable changes to the Bitmagnet qBittorrent Search Plugin will be documented in this file.

## [1.0] - 2024-01-30

### Initial Release

- GraphQL API integration with Bitmagnet
- Support for standard qBittorrent categories (all, movies, tv, anime, music, books, games, software, pictures)
- JSON configuration system
- Pagination support
- Category filtering
- Error handling with stderr output
